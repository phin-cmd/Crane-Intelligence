### Backup Database
```bash
docker exec crane-intelligence-db-1 pg_dump -U crane_user crane_intelligence > backup.sql
```

### Restore Database
```bash
cat backup.sql | docker exec -i crane-intelligence-db-1 psql -U crane_user crane_intelligence
```

### View Logs
